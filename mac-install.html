macOS · VS Code · 一行指令搞定

歡迎來裝 Claude Code！（Mac 版） 別緊張，第一次也沒關係～ 這一頁專門給用 Mac 的你，一樣一步一步來 🍎

主教材是給 Windows 電腦用的，如果你是 Mac，跟著這一頁做就好——一樣不用先懂程式，打開「終端機」貼一行指令、按 Enter，Claude Code 就會自己裝好。

開始前 · 溫馨提醒

麻煩你先準備好一個 Claude 訂閱，等一下才能順利使用喔 🙂 Claude Code 需要 Claude Pro 或 Max 方案（大約 US$20/月起）。安裝本身不用登入，但如果還沒訂閱，裝好後在登入那步會沒辦法用。建議先到 claude.ai 完成訂閱，再回來跟著下面做。

Terminal（終端機）
$ curl -fsSL https://claude.ai/install.sh | bash

還不知道「終端機」是什麼嗎？別擔心，往下看第一段，我們會慢慢說明給你聽 👇

前置

先認識「終端機」（Terminal，很簡單，別怕）

終端機是 Mac 本來就內建的小工具（一個可以打字下指令的視窗），不用另外安裝。打開它只要幾秒鐘，下面兩種方法任選一種就好：

方法一 · 最推薦
  1. 按鍵盤的 ⌘ Command + 空白鍵，叫出 Spotlight 搜尋
  2. 直接輸入 terminal
  3. 看到「終端機 Terminal」，按 Enter 打開就好
方法二 · 從資料夾找
  1. 打開 Finder → 左側點 應用程式 Applications
  2. 點進 工具程式 Utilities 資料夾
  3. 雙擊「終端機 Terminal」圖示

💡 打開後會看到一個框，游標在你的電腦名稱後面閃爍（長得像 yourname@MacBook ~ %），就表示準備好囉，可以貼指令了。貼上後按 Enter 就會開始，很簡單吧 🙂

流程

接下來，你只要做這 3 件事

1

打開終端機

用上面教的方法打開它，準備好就可以了，不用緊張。

2

貼上指令、按 Enter

把這一行貼上、按 Enter，然後放輕鬆，讓它自己跑就好。

curl -fsSL https://claude.ai/install.sh | bash
3

重開終端機、輸入 claude

裝好後關掉終端機、重新打開一個新的，輸入 claude、按 Enter，登入你的帳號，就完成啦！

安裝 VS Code

順手裝一個好用的編輯器

Claude Code 本身在終端機就能用，但搭配 VS Code 這個編輯器，你可以同時看到檔案、終端機、Claude 的對話畫面，會順手很多。這一步跟 Windows 版做法幾乎一樣：

1

下載 VS Code

到 code.visualstudio.com，選 Mac 版下載，拿到一個 .zip 檔。

2

拖進「應用程式」

解壓縮後把 Visual Studio Code 拖進 Finder 的「應用程式 Applications」資料夾。

3

裝擴充套件

打開 VS Code，左側「擴充功能」圖示（四個小方塊）→ 搜尋 Claude Code → 點安裝。

🍎

第一次打開可能會被擋：Mac 對「從網路下載的 App」比較保護，如果跳出「無法打開，因為來自身分不明的開發者」，去 系統設定 → 隱私權與安全性，捲到最下面會看到一個「強制打開」的按鈕，點一下就好。

備用

一鍵安裝沒成功？照著這樣手動裝一次

絕大部分 Mac 用一鍵指令都會順利，但如果你的電腦環境比較特殊，這裡有另外兩種官方認可的做法：

最推薦

① 官方原生安裝

就是上面那一行，適合所有人

curl -fsSL https://claude.ai/ install.sh | bash
Mac 老手愛用

② Homebrew

如果你電腦已經有裝 Homebrew

brew install --cask claude-code
傳統做法

③ npm 路線

需要先裝 Node.js 18 以上

npm install -g @anthropic-ai/claude-code
💡

不確定選哪個？就用①，那是官方現在最推薦的方式，不需要你電腦先裝任何東西，之後也會自動更新。②和③是給已經有自己開發習慣的人用的，一般人不用特別選。

卡關

遇到狀況？別擔心，這裡都有解

卡住的時候，不用自己對照下面的表格猜——最快、最準的方法是這一招：

📸

把整個錯誤畫面截圖，直接問 AI。按 ⌘ Command + Shift + 4 截圖（框選你要的範圍），打開手機或另一台電腦上的 claude.ai（網頁版就能用，不用等這台電腦先裝好），把截圖貼上去，跟它說：「我在 Mac 裝 Claude Code 卡住了，這是什麼意思？我該怎麼辦？」——它看得懂畫面上的文字，會直接告訴你下一步。

如果你想自己先查，這幾個是最常見的狀況：

狀況這樣做就好
打 curl 說找不到指令幾乎不可能發生——curl 是 Mac 內建的，如果真的找不到，代表你的 Mac 系統版本非常舊，先更新 macOS
打 claude 說找不到指令把終端機關掉，重新打開一個新視窗就好（讓設定生效）；還是不行，把終端機完全結束（Command+Q）再開一次
VS Code「無法打開，開發者身分不明」系統設定 → 隱私權與安全性 → 捲到最下面 → 按「強制打開」

裝完 Claude Code 沒反應去終端機打 echo $PATH，看有沒有出現 .local/bin，沒有的話重開機再試一次
完成
