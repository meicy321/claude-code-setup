Welcome to installing Claude Code! (Mac Edition) No stress, even if it's your first time — this page is just for Mac users, same gentle step-by-step 🍎

The main guide is written for Windows. If you're on a Mac, follow this page instead — you still don't need to know how to code. Just open the "Terminal," paste one command, press Enter, and Claude Code installs itself.

Before you start

Please have a Claude subscription ready first so everything works smoothly in a moment 🙂 Claude Code needs a Claude Pro or Max plan (roughly US$20/month and up). The install itself doesn't need a login, but you'll need it at the login step once it's installed. We recommend subscribing at claude.ai first, then coming back to follow along below.

Terminal
$ curl -fsSL https://claude.ai/install.sh | bash

Not sure what "Terminal" is yet? Don't worry — read the first section below and we'll explain it gently, step by step 👇

Prep

Meet the Terminal first (it's easy, don't be scared)

Terminal is a little tool that's already built into your Mac (a window where you can type commands) — nothing extra to install. Opening it takes just a few seconds. Pick whichever of the two ways below you like:

Method 1 · Recommended
  1. Press ⌘ Command + Space together to bring up Spotlight search
  2. Type terminal
  3. When you see "Terminal," press Enter to open it
Method 2 · From Finder
  1. Open Finder → click Applications in the sidebar
  2. Open the Utilities folder
  3. Double-click the "Terminal" icon

💡 Once it opens you'll see a window with a blinking cursor after your computer's name (something like yourname@MacBook ~ %) — that means it's ready and you can paste the command. Paste it, press Enter, and it'll get going. Easy, right? 🙂

Flow

Next, all you have to do is these 3 things

1

Open Terminal

Open it using the method above. Once it's ready you're good to go — no need to be nervous.

2

Paste the command, press Enter

Paste this line, press Enter, then relax and let it run on its own.

curl -fsSL https://claude.ai/install.sh | bash
3

Reopen Terminal, type claude

Once it's installed, close Terminal and open a fresh one, type claude, press Enter, log in, and you're done!

Behind the scenes

The Mac edition uses the official "native install" route — no Node.js required, no Homebrew required. This command downloads a self-contained program. Here's how it differs from the Windows edition:

The biggest difference from Windows: your Mac's built-in Terminal (zsh) already gets along fine with Claude Code — you don't need the extra Git Bash setup step the Windows edition requires. One less thing to worry about; call it a small Mac-user perk 🙂

Install VS Code

A nice editor to go with it

Claude Code works fine in Terminal on its own, but pairing it with VS Code lets you see your files, the terminal, and Claude's conversation all at once — much smoother. This step is nearly identical to the Windows edition:

1

Download VS Code

Go to code.visualstudio.com, pick the Mac version, and download the .zip file.

2

Drag into Applications

After unzipping, drag Visual Studio Code into Finder's "Applications" folder.

3

Install the extension

Open VS Code, click the Extensions icon in the sidebar (four little squares) → search for Claude Code → click Install.

🍎

First launch might get blocked: Macs are protective of apps downloaded from the internet. If you see "can't be opened because it is from an unidentified developer," go to System Settings → Privacy & Security, scroll to the bottom, and click "Open Anyway."

Backup plan

One-click install didn't work? Here's how to do it by hand

The one-click command works smoothly for the vast majority of Mac users, but if your setup is a bit unusual, here are two other officially supported methods:

Recommended

① Official native install

Same as above — works for everyone

curl -fsSL https://claude.ai/ install.sh | bash
Mac power users

② Homebrew

If you already have Homebrew installed

brew install --cask claude-code
Legacy route

③ npm route

Requires Node.js 18 or higher first

npm install -g @anthropic-ai/claude-code
💡

Not sure which to pick? Use ①. It's the officially recommended way, doesn't require anything pre-installed on your Mac, and keeps itself updated afterward. ② and ③ are for people who already have their own development habits — most people don't need to choose them specially.

Stuck?

Hit a snag? Don't worry, there's a fix here

When you're stuck, don't try to match your situation against the table below — the fastest, most reliable move is this one:

📸

Screenshot the whole error screen and just ask AI. Press ⌘ Command + Shift + 4 to screenshot it (drag to select the area), open claude.ai on your phone or another computer (the web version works fine — you don't need this machine set up already), paste the screenshot in, and say: "I'm stuck installing Claude Code on my Mac — what does this mean, and what should I do?" It can read the text on your screen and will tell you exactly what to do next.

If you'd rather look it up yourself first, here are the most common issues:

SituationDo this
curl: command not foundAlmost never happens — curl ships with every Mac. If it truly doesn't exist, your macOS version is very old; update macOS first
claude: command not foundClose Terminal and open a brand-new window (so the settings take effect); if it still doesn't work, fully quit Terminal (Command+Q) and reopen it
VS Code "can't be opened, developer cannot be verified"System Settings → Privacy & Security → scroll to the bottom → click "Open Anyway"
Permission deniedThe official installer never asks for your password; if something does prompt for one, screenshot it and ask AI whether that's expected
Nothing happens after installing Claude CodeIn Terminal, run echo $PATH and check whether .local/bin appears; if not, restart your Mac and try again
