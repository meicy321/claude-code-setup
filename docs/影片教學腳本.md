# 免驚 AI ｜ Claude Code 安裝教學影片腳本

> 用途：照這份分鏡＋旁白，用螢幕錄影軟體（OBS、PowerPoint 錄製、Loom 等）錄一支約 2.5～3 分鐘的教學影片。
> 語氣：親切、溫和、鼓勵，把新手當朋友。
> 建議：畫面用你自己的電腦實際操作錄，旁白照下面唸。片頭片尾可放「免驚 AI」品牌。

---

## 🎬 分鏡表（Scene by Scene）

### 0. 片頭（0:00–0:08）
- **畫面**：黑底或珊瑚橘底，中央「免驚 AI」logo 字樣，下方小字「AI 免驚，跟著做就會」。
- **旁白**：
  > 「哈囉，歡迎來到免驚 AI。今天我帶你用一行指令，把 Claude Code 裝好，全程不用怕，跟著做就會。」

### 1. 先講前提（0:08–0:20）
- **畫面**：螢幕顯示 claude.ai 訂閱頁（Pro / Max 方案）。
- **旁白**：
  > 「開始之前提醒一件事：要實際使用 Claude Code，需要先有 Claude Pro 或 Max 的訂閱。安裝本身不用登入，但用之前記得先辦好訂閱喔。」

### 2. 打開 PowerShell（0:20–0:35）
- **畫面**：按左下角 Windows 開始鍵 → 輸入「powershell」→ 點開「Windows PowerShell」。
- **旁白**：
  > 「第一步，打開 PowerShell。按鍵盤左下角的開始鍵，直接打 powershell，點開它就好。它是 Windows 內建的，不用另外裝。」

### 3. 貼上那一行指令（0:35–0:55）
- **畫面**：把安裝指令貼上、游標停在指令上。
  ```
  irm https://raw.githubusercontent.com/meicy321/claude-code-setup/main/install-claude-code.ps1 | iex
  ```
- **旁白**：
  > 「接著，把這一行指令貼上去，按 Enter。就這樣，剩下的它會自動幫你處理。是不是很簡單？」

### 4. 幫專案取名字（0:55–1:08）
- **畫面**：腳本問「想叫什麼名字？」，示範打一個名字（例如 my-first-app）或直接按 Enter。
- **旁白**：
  > 「它會問你，要幫專案資料夾取什麼名字。你可以自己取一個，或直接按 Enter 用預設的就好。這個資料夾，就是等一下 Claude 幫你工作的地方。」

### 5. 自動安裝（1:08–1:30）
- **畫面**：快轉／加速播放安裝過程，一路出現綠色 [OK]（Git、Node、Claude Code、VS Code、擴充套件）。
- **旁白**：
  > 「接下來它會自動把需要的東西一個一個裝好——Git、Node、Claude Code，還有 VS Code 跟編輯器裡的擴充套件。你只要看著綠色的 OK 一路出現，就代表一切順利。放輕鬆，等它跑完就好。」

### 6. 桌面出現圖示（1:30–1:45）
- **畫面**：安裝完成畫面 → 切到桌面，出現「Claude Code」圖示 → 滑鼠雙擊它。
- **旁白**：
  > 「裝好之後，桌面會出現一個 Claude Code 的圖示。以後要用，直接雙擊它就好。我們現在就點兩下。」

### 7. VS Code 打開專案 + 開終端機（1:45–2:15）
- **畫面**：VS Code 打開專案資料夾 → 滑鼠移到最上方選單列 →「Terminal」→「New Terminal」→ 終端機從下方跳出（Git Bash）。
- **旁白**：
  > 「它會用 VS Code 打開你的專案。接下來我們要打開終端機：看畫面最上方的選單列，點 Terminal，再點 New Terminal。你會看到終端機從畫面下方跳出來，這個終端機我們已經幫你設好用 Git Bash 了。」

### 8. 輸入 claude、登入（2:15–2:40）
- **畫面**：在終端機打「claude」按 Enter → 跳出瀏覽器登入 → 授權完成 → 回到 Claude Code 歡迎畫面。
- **旁白**：
  > 「在終端機裡輸入 claude，按 Enter。第一次它會自動打開瀏覽器，請你登入。用你的 Claude 帳號登入、授權，完成後回到這裡——看，Claude Code 就開始工作啦！」

### 9. 片尾（2:40–2:55）
- **畫面**：Claude Code 運作畫面 → 淡出到「免驚 AI」品牌卡，顯示：
  ```
  免驚 AI ｜ AI 免驚，跟著做就會
  更多教學：github.com/meicy321/claude-code-setup
  ```
- **旁白**：
  > 「恭喜你，你已經裝好 Claude Code 了，真的不難吧？我是免驚 AI，我們下支影片見，掰掰！」

## 🛠️ 錄製小提醒
- 螢幕錄影推薦：OBS Studio（免費）、或 Windows 內建 Xbox Game Bar（Win+G）。
- 安裝過程較長的段落（第 5 段）可以剪接／加速，不用完整播完。
- 游標移動放慢一點、點擊的地方停一下，新手比較跟得上。
- 片頭片尾的「免驚 AI」品牌卡，可以用 Canva 做一張圖。
