## 今天完成的事

### 1. 解析參考教材
- 用 ffmpeg 抽影格 + 讀燒錄字幕，看完「矽谷叔叔」HiSKIO《Claude Code 安裝與環境準備（Windows）》。
- 確認課程流程：Git Bash + Node + `npm install -g` → 裝 VS Code → 裝「Claude Code for VS Code」擴充 → VS Code 預設終端機選 Git Bash → 終端機打 `claude`。
- 對照官方文件，改用更穩的做法：**Claude Code 本體走原生安裝，不用 npm**（自動更新、不被 Node 綁死）。

### 2. 一鍵安裝腳本 `install-claude-code.ps1`
- winget 裝 Git、Node.js LTS、**VS Code**、**擴充套件 anthropic.claude-code**。
- Claude Code 本體用官方 `install.ps1` 原生安裝（有 winget 備援）。
- 自動把 VS Code 預設終端機設成 **Git Bash**（合併寫入 settings.json，不覆蓋既有設定）。
- 建立**專案資料夾**（可自訂命名，直接 Enter 用預設 claude-code）+ **桌面「Claude Code」捷徑**（雙擊用 VS Code 打開專案）+ 放一份「先看我-如何開始.txt」。
- 檢查 Windows 版本、winget、TLS 1.2；全程可重複執行、有防呆。
