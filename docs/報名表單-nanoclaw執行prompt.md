**第 4 題**

- 題型：簡答
- 必填：是

**第 6 題**
- 題目：請問您想來報名這堂課的原因是？想解決什麼工作或生活上的問題呢？
- 題型：段落（長文）
- 必填：是

**第 7 題**
- 題目：現場成果展示會秀幾個比較大型的作品，想先問問您對哪一個最有興趣（會依大家的興趣調整現場多講哪一個）
- 題型：單選題
- 必填：是
- 選項：
  1. 📚 個人知識庫／筆記整理術（把零散的對話、筆記自動變成一座有系統的知識庫）
  2. 🎮 給小孩玩的英語學習遊戲（單字、文法練習，可愛配色、收集進度）
  3. 🦉 AI 陪讀教練（用語音陪小孩讀英文故事書、即時糾正發音）
  4. 都很有興趣，都想多聽一點

**第 8 題**
- 題目：系統確認：請問您上課當天攜帶的筆記型電腦，是 Windows 系統嗎？
- 題型：單選題
- 必填：是
- 選項：
  1. 是，我是 Windows 筆電（可攜帶至現場）
  2. 不是（本場次僅限 Windows 系統，若為其他系統，恕無法參加本場次）

**第 9 題**
- 題目：硬體規格檢查：請問您的電腦記憶體 (RAM) 大小是多少？
- 說明：💡【RAM 快速查詢 3 步驟】① Windows 鍵 + I → 設定 ② 點「系統」→ 滑到最下面「關於」③ 找「已安裝 RAM」，後面的數字就是（例如 8.00 GB 或 16.00 GB）。不確定也沒關係，選最後一個選項就好。
- 題型：單選題
- 必填：是
- 選項：
  1. 8GB（基本要求，開課前建議關閉背景其他軟體）
  2. 16GB (含) 以上（優質推薦，運算速度流暢）
  3. 看了步驟還是不太確定，但我當天會帶 Windows 筆電過去，提早 10 分鐘到現場當場確認

**第 10 題**
- 題目：課前準備確認：請問您是否已預先註冊好 Claude 帳號並升級至 Pro 付費版？
- 題型：單選題
- 必填：是
- 選項：
  1. 是，已經準備好了！
  2. 還沒，但我會在課前完成註冊與升級（Pro 版本）。

**第 11 題**
- 題目：備註或想對我說的話
- 題型：段落（長文）
- 必填：否
