開始進前 · 溫馨提醒

拜託你先準備好一个 Claude 注文，等一下才會當順利使用喔 🙂 Claude Code 需要 Claude Pro 抑是 Max 方案（差不多 US$20/月起）。安裝本身毋免 log-in，毋過若是猶未注文，裝好了後佇 log-in 彼步就袂當用。建議先去 claude.ai 完成注文，才閣倒轉來綴下跤做。

Terminal（終端機）
$ curl -fsSL https://claude.ai/install.sh | bash

猶毋知「終端機」是啥物嗎？免煩惱，繼續看下跤第一段，阮會勻勻仔講予你聽 👇

前置

先來捌「終端機」（Terminal，足簡單，莫驚）

終端機是Mac原本就有的軟體（一个會當拍字落指令的視窗），毋免另外安裝。拍開伊只要幾秒鐘，下跤兩種方法揀一種就好：

方法一 · 上推薦
  1. 㧎鍵盤的 ⌘ Command + 空白鍵，叫出 Spotlight 搜尋
  2. 直接拍 terminal
  3. 看著「終端機 Terminal」，㧎 Enter 拍開就好
方法二 · 對資料夾揣
  1. 拍開 Finder → 倒爿㧎 應用程式 Applications
  2. 㧎入去 工具程式 Utilities 資料夾
  3. 雙㧎「終端機 Terminal」圖示

💡 拍開了後會看著一个框，游標佇你電腦的名後壁閃爍（生做親像 yourname@MacBook ~ %），就表示準備好矣，會當貼指令矣。貼起來了後㧎 Enter 就會開始，足簡單的齁 🙂

流程

接落來，你只要做這 3 項代誌

1

拍開終端機

用頂懸教的方法拍開伊，準備好就會使矣，毋免緊張。

2

貼指令、㧎 Enter

共這逝貼起去、㧎 Enter，然後放輕鬆，予伊家己走就好。

curl -fsSL https://claude.ai/install.sh | bash
3

重開終端機、拍 claude

裝好了後共終端機關掉、重新拍開一个新的，拍 claude、㧎 Enter，log-in 你的帳號，就完成矣！

安裝 VS Code

順紲裝一个好用的編輯器

Claude Code 本身佇終端機就會使用，毋過搭配 VS Code 這个編輯器，你會當同齊看著檔案、終端機、Claude 的對話畫面，會較順手：

1

下載 VS Code

去 code.visualstudio.com，揀 Mac 版下載，提著一个 .zip 檔。

2

拖入去「應用程式」

解壓縮了後共 Visual Studio Code 拖入去 Finder 的「應用程式 Applications」資料夾。

3

裝擴充套件

拍開 VS Code，倒爿「擴充功能」圖示（四塊細方塊）→ 搜尋 Claude Code → 㧎安裝。

🍎

第一擺拍開可能會予擋著：Mac 對「對網路下載的 App」較保護，若是迸出「無法度拍開，因為是身份不明的開發者」，去 系統設定 → 隱私權佮安全性，捲到上下面會看著一个「強制拍開」的按鈕，㧎一下就好。

備用

一鍵安裝無成功？照按呢手動裝一擺

大部份 Mac 用一鍵指令攏會順利，毋過若是你的電腦環境較特殊，遮有另外兩種官方認可的做法：

上推薦

① 官方原生安裝

就是頂懸彼逝，適合逐家

curl -fsSL https://claude.ai/ install.sh | bash
Mac 老手愛用

② Homebrew

若是你電腦已經有裝 Homebrew

brew install --cask claude-code
傳統做法

③ npm 路線

需要先裝 Node.js 18 以上

npm install -g @anthropic-ai/claude-code
💡

無確定欲揀佗一个？就用①，彼是官方這馬上推薦的方式，毋免你電腦先裝任何物件，了後嘛會家己更新。②佮③是予已經有家己開發習慣的人用的，一般人毋免特別揀。
