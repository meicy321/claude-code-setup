Claude Code 無腦安裝 複製一行、貼上、按 Enter — 環境全自動裝好

不用自己一個一個下載 Node.js、Git、判斷版本。腳本會偵測你的電腦，把該裝的都裝好，你只要照著綠字走到底。

Windows PowerShell
PS> irm https://raw.githubusercontent.com/meicy321/claude-code-setup/main/install-claude-code.ps1 | iex

※ 這行在 repo 上傳到 github.com/meicy321/claude-code-setup 後即可使用，直接發給學生。

流程

學生只要做 3 件事

1

打開 PowerShell

按 開始 鍵，輸入 PowerShell，點開它。

2

貼上指令、按 Enter

貼上老師給的那一行，按 Enter，接著就放著讓它自己跑。

3

重開視窗、登入

跑完 → 關掉重開 PowerShell → 打 claude → 瀏覽器登入。

卡關

不同 Windows 版本常見問題

狀況怎麼修
找不到 winget開 Microsoft Store 搜「App Installer」更新，或到 aka.ms/getwinget
Windows 太舊（低於 10 1809）先更新 Windows，或改用 WSL 安裝
下載一直失敗 / TLS 錯誤腳本已自動強制 TLS 1.2；手動時先跑 [Net.ServicePointManager]::SecurityProtocol='Tls12'
打 claude 說找不到指令關掉 PowerShell，重開一個新視窗（讓 PATH 生效）
有裝 Git 但找不到 Bash在 %USERPROFILE%\.claude\settings.json 設 CLAUDE_CODE_GIT_BASH_PATH
🔑

裝完最後一步：登入
